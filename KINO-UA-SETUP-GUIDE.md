# 🎬 КІНО.UA — TMDB Інтеграція та Налаштування

## 📋 Кроки для отримання TMDB API ключа

### 1️⃣ Реєстрація на themoviedb.org
- Перейди на https://www.themoviedb.org/signup
- Заповни форму реєстрації (Email, Username, Password)
- Підтверди email

### 2️⃣ Отримання API ключа
- Зайди до **Settings** → **API** (у налаштуваннях акаунту)
- Натисни **Create** для нового API ключа
- Обери тип: **Developer**
- Прочитай і прийми умови
- Отримай **API Key (v3 auth)**

### 3️⃣ Використання у додатку

#### Варіант 1: Через веб-інтерфейс (НАЙПРОСТІШИЙ)
1. Відкрий сайт: https://kino-ua-server-ready.vercel.app/
2. Натисни іконку ⚙️ (Settings) у верхньому правому куті
3. Вставь TMDB API ключ у поле
4. Натисни **Зберегти**
5. Готово! Фільми завантажуватимуться автоматично

#### Варіант 2: Через .env файл
1. У корені проекту створи/відредагуй файл `.env.local`
2. Додай рядок:
```
NEXT_PUBLIC_TMDB_API_KEY=ТИ_АПІКЛЮЧ_СЮДИ
```
3. Перезавантажь додаток

#### Варіант 3: Vercel (для хостингу на Vercel)
1. Зайди до проекту на Vercel: https://vercel.com/
2. Перейди до **Settings** → **Environment Variables**
3. Додай змінну:
   - Name: `NEXT_PUBLIC_TMDB_API_KEY`
   - Value: `твій TMDB API ключ`
4. Redeploy проект

## 🚀 Як розгорнути на Vercel без GitHub

### Способ 1: Vercel CLI (НАЙСКОРИЙ)
```bash
npm install -g vercel
vercel login
vercel deploy
```

### Способ 2: Drag & Drop на Vercel
1. Зайди на https://vercel.com/
2. Натисни "Create New Project"
3. Перетягни папку `kino-ua` у вікно браузера
4. Додай TMDB API ключ як Environment Variable
5. Deploy!

### Способ 3: Вручну оновлювати (Git необов'язковий)
1. Внеси зміни локально
2. Запусти `vercel deploy`
3. Скопіюй посилання
