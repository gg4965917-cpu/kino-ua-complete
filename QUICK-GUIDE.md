# 🎬 КІНО.UA — Швидкий Довідник

## 🔥 10 Питань, які часто задають

### Q1: Як додати свій TMDB API ключ?
**A:** Натисни ⚙️ у верхньому правому куті → вставь ключ → "Зберегти"

### Q2: Де взяти TMDB API ключ?
**A:** https://www.themoviedb.org/settings/api
```
1. Заходиш на themoviedb.org/signup
2. Реєструєшся
3. Settings → API
4. Create → Developer
5. Копіюш API Key (v3)
```

### Q3: Це платно?
**A:** НІ! TMDB API та основні функції БЕЗКОШТОВНІ
- Базовий доступ до 500K+ фільмів = безплатно
- Бізнес планів з додатковими фічами = платно

### Q4: Як розгорнути на Vercel?
**A:** 
```bash
npm install -g vercel && vercel login && vercel deploy
```

### Q5: Чому не завантажуються фільми?
**A:** 
- ❌ Неправильний/порожній API ключ → додай в Settings
- ❌ Немає інтернету → перевір з'єднання
- ❌ VPN блокує TMDB → спробуй без VPN
- ✅ Без ключа показуються 10 українських фільмів

### Q6: Як зберегти фільм у улюбленні?
**A:** Натисни ❤️ на карточці фільму або у деталях

### Q7: Як дивитись фільм?
**A:** Натисни "Дивитись" → запуститься плеєр (демо режим поки)
Наразі це симуляція, реальний відео потребує VideoCDN/Ashdi

### Q8: Де див історія перегляду?
**A:** Категорія "Переглянуті" → там усі фільми що ти дивився

### Q9: Як відключити TMDB?
**A:** ⚙️ → натисни "Відключити" → повертаються 10 українських фільмів

### Q10: Чи можу я зберегти свої дані?
**A:** ДА! Все зберігається локально в браузері (localStorage)
- Улюблені фільми
- Історія перегляду
- Твої рейтинги
- Прогрес перегляду

## 🎯 Основні функції

### 🔍 ПОШУК
- Вводь назву фільму у пошук
- Динамічні підказки в реальному часі
- Пошук працює у фільмах TMDB

### 🏷️ ФІЛЬТРИ
- **За жанрами**: Драма, Бойовик, Sci-Fi тощо
- **За сортуванням**: Рейтинг, Рік, Назва, Популярність, Моя оцінка
- **Озвучка**: Тільки фільми з українською озвучкою

### ❤️ УЛЮБЛЕНІ
- Натисни ❤️ щоб додати/видалити
- Категорія "Мій список" → усі улюблені
- Зберігаються локально в браузері

### ⭐ МОЇ ОЦІНКИ
- Натисни на зірки у деталях фільму
- 1-5 зірок = твоя оцінка
- Зберігаються автоматично

### 📺 ПРОДОВЖИТИ ПЕРЕГЛЯД
- Коли почнеш дивитися фільм, система відслідковує прогрес
- Блок "Продовжити перегляд" покаже твої незавершені фільми
- Натисни щоб продовжити з тієї ж позиції

### 🌙 ВНОЧІ/СВІТЛЕ
- Dark Mode заходомо увімкнений (чорний фон)
- Персонаж автоматично підлаштовується під час дня

## 🔐 Приватність та Безпека

### Де зберігаються дані:
- **Локально на твоєму ПК**: улюблені, рейтинги, історія
- **На TMDB серверах**: інформація про фільми (публічна)
- **Ніде не зберігається**: хист TMDB API ключа (берегти!)

### Безпека API ключа:
- ❌ НІКОЛИ не коміти ключ у Git
- ❌ НІКОЛИ не показуй ключ в соцмережах
- ✅ Використовуй .env.local (в .gitignore)
- ✅ На Vercel використовуй Environment Variables
