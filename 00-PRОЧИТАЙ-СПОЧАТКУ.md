## ⚡ 3 Кроки до Запуску

### Крок 1️⃣: Отримай TMDB API ключ (2 хвилини)
```
👉 Переходь на https://www.themoviedb.org/signup
1. Заповни форму (Email, Username, Password)
2. Підтвердь email
3. Зайди у Settings → API
4. Натисни "Create" → "Developer"
5. Скопіюй "API Key (v3 auth)" ← ЦЕ ТОБІ ПОТРІБНО!
```

### Крок 2️⃣: Розгорни на Vercel (30 секунд)
```bash
# Встановлюємо Vercel
npm install -g vercel

# Логуємось
vercel login

# Переходимо у папку проекту
cd kino-ua-updated

# Розгортаємо!
vercel deploy
```

### Крок 3️⃣: Додай TMDB ключ (1 хвилина)
```
Коли сайт завантажиться:
1. Натисни ⚙️ (Settings) у верхньому правому куті
2. Вставь TMDB API ключ у поле
3. Натисни "Зберегти та підключити"
4. ✅ Готово! Фільми завантажуватимуться автоматично
```

## 🆘 Щось Не Працює?

### Фільми не завантажуються?
1. Перевір інтернет
2. Перевір що ключ скопійовано без пробілів
3. Перевір що це v3 API key (не v4 Bearer Token)
4. F12 → Console → подивись помилки

### Сайт не завантажується?
1. Перевір що `npm install` закінчився успішно
2. Перевір що `npm run dev` запущений
3. Спробуй清清 браузер кеш
4. Перезавантажимо http://localhost:3000

### API ключ невірний?
1. Перейди на https://www.themoviedb.org/
2. Settings → API
3. Отримай новий ключ
4. Скопіюй точно (без пробілів!)

## 🎨 Основні Функції

### 🔍 Пошук
- Вводь назву фільму
- Побачиш результати у реальному часі
- Натисни на результат щоб перейти до деталей

### ❤️ Улюблені
- Натисни ❤️ на карточці
- Вони з'являтимуться у категорії "Мій список"
- Зберігаються автоматично

### ⭐ Оцінювання
- Відкрий деталі фільму
- Натисни на зірки (1-5)
- Оцінка зберігається

### 📺 Історія
- Категорія "Переглянуті"
- Показує всі переглянуті фільми
- З прогресом перегляду
